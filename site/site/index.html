TOS

TOS is an open source linux distribution that I made and activally maintain. The core focus of this OS is easy and fast development. It does this by beeing extremly efficient and using low resources without compormising usability. It has a lot of features but the most important are: LUKS encryption, i3 as a window manager, wayland and x11 support and much more. The system is easily maintainable thanks to the tos cli tool and is always up to date with upstream.

Project Zeppelin

Project zeppelin is a universal cheat sheet build for quick lookup. This project helps developers to quickly find language features. Every developer forgets how to write a specific piece of code. This cheat sheet is the ideal lookup table. It features design patterns, language syntax, framework style code, git commands and much more. This project is build for both the web and android in corporation with Matthias Willemsen.

Automatic Sentry Gun

As a thesis for high school I build an automated sentry gun. This involves detecting movement and moving the sentry gun towards the moving person. This is achieved using computer vision and translating the pixel coordinates to real motor movement. When the turrent has aimed to the person it proceeds shooting paintball bullets at the person. Ideal for playing paintball.
